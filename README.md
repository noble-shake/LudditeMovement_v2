# Repository Name : LudditeMovement_v2
# Soul Heroes 프로젝트

플레이어 오브 (마우스 포인터를 따라가는 오브젝트)가 적의 탄막을 피하고, 소울 게이지를 사용하여 영웅 플레이어들을 깨워 적들을 물리치는 탄막 슈팅과 SRPG를 결합한 게임 입니다.

1. Custom Editor를 이용하여 저장/로드, 맵 제작, 스테이지 구성
2. 오브젝트 풀링 및 게임의 모든 데이터 저장/불러오기
3. 플레이어를 조종 (A* PathFinding)하는 포인터와 상호작용 구현
4. 스킬 트리 UGUI 작업
